# 🎓 Smart Curriculum & Attendance Management System

A full-stack web application for college attendance management using QR codes and GPS verification.
made with assistence of claude


## 👥 Roles
- 🎓 Student — Mark attendance, view reports
- 👨‍🏫 Faculty — Generate QR, manage syllabus
- 👩‍💼 Class Advisor — Monitor section students
- 👨‍💼 HOD/Admin — Full system control

## ✨ Features
- QR Code based attendance
- GPS verification
- Late attendance detection
- Weekly timetable
- Syllabus coverage tracking
- Real-time reports
- Low attendance alerts

## 🛠️ Tech Stack
- **Backend:** Java, Spring Boot, JWT, MySQL
- **Frontend:** React.js, Axios
- **Tools:** IntelliJ, VS Code, Postman

## 🚀 Run Locally

### Backend
```bash
cd backend
mvn spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm start
```
